# 改进实验:能否超越冠军?(样本外 OOS)

> 2026-06-29。脚本:`improve.py`(OOS 训练+评测)、`pnl_sim.py`(PnL 模拟)、`eval_harness.py`(准确率评测)。均为新增,不动原仓库文件。

## 背景:官方「模型评分」的真相
逆向 `私榜结果_2601008.csv` 发现:**模型评分 ≈ 由 `pnl_average`(单次收益率)主导**(corr 0.71),且与 recall/f_score **负相关**。
→ 赛制奖励"每次出手的平均收益",出手越多越吃亏。冠军(user-2601008)正是靠极度精选、单次收益率全场最高夺冠(私榜 model_score 第 1/26)。

## 实验设计(公平 OOS)
- 按 **date 切分**:train=0–44 偶数日、val=46–55、test=56–78(模型**没见过** test)。
- 对照口径:**单次收益率 pnl_average**(按预测方向吃未来 N tick 中间价变动,除以出手数)。
- 阈值在 val 上为每个 horizon 选最优,再上 test(防止用 test 调参)。
- 两套特征各自重训 5 个 XGBoost(multi:softprob, depth6, 200 轮):
  - **base**:原版 `build_features`(冠军同款特征)
  - **plus**:base + 微观结构因子

## 新增的微观结构因子(`improve.py: extra_features`)
- **micro-price**:量加权中间价 `(ask·bidsz+bid·asksz)/(bidsz+asksz)` 及其与中间价之差 —— 短线方向强预测
- **OFI 订单流不平衡**:买一/卖一价量变动近似 + 5/20 滑动均值
- **多档加权队列不平衡**:5 档量按 1.0/0.8/.../0.2 加权的 bid-ask imbalance
- **价差动量**:spread 一阶差分

## 结果(OOS test)

| horizon | base 单次收益 | plus 单次收益 |
|---|---|---|
| label_5  | 0.002144 | 0.002348 |
| label_10 | 0.002700 | 0.002605 |
| label_20 | 0.002606 | 0.003166 |
| label_40 | 0.002732 | **0.004284** |
| label_60 | 0.001312 | 0.001644 |
| **均值** | **0.002299** | **0.002809 (+22%)** |

- val 与 test 数值接近 → 非过拟合,改进可泛化。
- label_40 提升最显著(+57%);label_10 基本持平。

## 深挖复核(deepdive.py)—— 诚实修正

上面 +22% 是**单次 split、seed=0 的乐观值**。多随机种子复核后:

**A. 因子消融(seed=0)**:base 0.002231 → +micro **+11%**(最强)/ +ofi +9% / +qimb +10% / 全加 +8%。
→ micro-price 单因子贡献最大;"全加"反不如单加(因子冗余),可剪枝。

**B. 稳健性(3 seeds)**:base 0.002357±0.000089 vs all 0.002580±0.000136 → **稳健提升 ≈ +9.5%**(3 种子里 all 全程 > base)。

**C. 门控对比**:max-prob 阈值(0.00242)> signed 净概率(0.00221)→ 原版门控更优,signed 无效。

## 结论(修正版)
**在公平样本外、多种子复核下,加入微观结构因子把官方核心指标(单次收益率)稳健提升约 +10%(单次 split 上可达 +22%,含运气)。** micro-price 是最有价值的单因子。仍属冠军同款方法之上的实质改进,但幅度应以 ~+10% 为准。

⚠️ 本地 OOS 近似口径,非官方 grader;闭门课程赛无法再上官方榜。

## 进阶实验 A/B/C/D(experiments_abcd.py / why_pertrade.py)

**A. 选股目标:单次收益 vs 夏普 —— 目标变了,最优策略反转**
| 优化目标 | 单次收益 | 出手数 | 总收益 | 夏普 |
|---|---|---|---|---|
| 最大化单次收益 | 0.002415 | 1,506 | 2.9 | 21.8 |
| 最大化夏普 | 0.000747 | **193,766** | 137 | **149** |
→ 夏普=边际×√出手数,逼你疯狂走量(19万笔);官方按单次收益,逼你精选。**指标定义策略。**
(注:夏普口径的19万笔忽略了交易成本;单次收益口径隐含了"只在边际>成本时出手",更贴近真实做市。)

**B. 因子剪枝**:base 0.002231 → +micro 0.002484 → **+micro+ofi 0.002578(最优)** → 全加 0.002415。
→ 砍掉 qimb 反而更好(它在稀释);**micro+ofi 是最优子集**。

**C. 模型对比**(全特征):XGBoost 0.002415 vs **LightGBM 0.002677(+11%)**。

**D. 容量曲线**(`capacity_curve.png`):总收益@阈值0.34(走量)/ 夏普@0.44(中间)/ 单次收益@0.84(精选)三处见顶——三个指标三个最优工作点。

**🏆 最优组合 = LightGBM + micro_ofi 剪枝 = 0.002834(比 base +27%)**
| 配置 | 单次收益 | vs base |
|---|---|---|
| base | 0.002231 | — |
| XGB+全微观 | 0.002415 | +8% |
| XGB+micro_ofi | 0.002578 | +16% |
| LGB+全特征 | 0.002677 | +20% |
| **LGB+micro_ofi** | **0.002834** | **+27%** |

## 结论汇总
- 决定排名的指标 = 单次收益率(超额),不是总收益/f-score(实证:总收益第1→榜21、f第1→垫底)。
- 最优配置:**LightGBM + (micro-price + OFI) 因子 + max-prob 阈值门控**,OOS 比 base +27%。
- micro-price 是最值钱的单因子;qimb 该剪;signed 门控无效。
- 目标函数决定策略:官方(单次收益)→ 精选;夏普 → 走量。选对指标比调模型更重要。
